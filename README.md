# Jokester

Jokester is a fun social media app centered around jokes. It allows users to submit and view jokes, view posts with comments, and rate jokes and comments submitted by others.

## Technologies
* Node JS
* Express JS
* MySQL Database
* Sequelize ORM
* Handlebars
* Passport
